Let $S$ be a collection of sets. A \emph{selector} of~$S$ is by definition a set $\{f(R)|R\in S\}$ where $f$ is a choice function on~$S$ (that is $f(R)\in R$).

There is a different selector of~$S$ for every choice function~$f$.
