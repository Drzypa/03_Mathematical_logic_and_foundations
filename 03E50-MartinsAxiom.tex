For any cardinal $\kappa$, \emph{Martin's Axiom for $\kappa$} ($MA_\kappa$) states that if $P$ is a partial order satisfying ccc then given any set of $\kappa$ \PMlinkname{dense}{DenseInAPoset} subsets of $P$, there is a \PMlinkname{directed}{DirectedSet} subset intersecting each such subset.  \emph{Martin's Axiom} states that $MA_\kappa$ holds for every $\kappa<2^{\aleph_0}$.
