beth numbers


The beth numbers are infinite cardinal numbers defined in a similar manner to the aleph numbers, as described below. They are written ℶα, where ℶ is beth, the second letter of the Hebrew alphabet, and α is an ordinal number.

We define ℶ0 to be the first infinite cardinal (that is, ℵ0). For each ordinal α, we define ℶα+1=2ℶα. For each limit ordinal δ, we define ℶδ=⋃α∈δℶα.

Note that ℶ1 is the cardinality of the continuum.

For any ordinal α the inequality ℵα⩽ℶα holds. The Generalized Continuum Hypothesis is equivalent to the assertion that ℵα=ℶα for every ordinal α.

For every limit ordinal δ, the cardinal ℶδ is a strong limit cardinal. Every uncountable strong limit cardinal arises in this way.
