proof of fixed points of normal functions


Suppose f is a κ-normal function and consider any α<κ and define a sequence by α0=α and αn+1=f⁢(αn). Let αω=supn<ω⁡αn. Then, since f is continuous,

f⁢(αω)=supn<ω⁡f⁢(αn)=supn<ω⁡αn+1=αω

So Fix⁡(f) is unbounded.

Suppose N is a set of fixed points of f with |N|<κ. Then

f⁢(sup⁡N)=supα∈N⁡f⁢(α)=supα∈N⁡α=sup⁡N

so sup⁡N is also a fixed point of f, and therefore Fix⁡(f) is closed.
