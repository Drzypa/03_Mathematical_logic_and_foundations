superset


Given two sets A and B, A is a superset of B if every element in B is also in A. We denote this relation as A⊇B. This is equivalent to saying that B is a subset of A, that is A⊇B⇔B⊆A.

Similar rules to those that hold for ⊆ also hold for ⊇. If X⊇Y and Y⊇X, then X=Y. Every set is a superset of itself, and every set is a superset of the empty set.

We say A is a proper superset of B if A⊇B and A≠B. This relation is sometimes denoted by A⊃B, but A⊃B is often used to mean the more general superset relation, so it should be made explicit when “proper superset” is intended, possibly by using X⊋Y or X⫌Y (or X⊋Y or X⫌Y).

One will occasionally see a collection C of subsets of some set X made into a partial order “by containment”. Depending on context this can mean defining a partial order where Y≤Z means Y⊆Z, or it can mean defining the opposite partial order: Y≤Z means Y⊇Z. This is frequently used when applying Zorn’s lemma.

One will also occasionally see a collection C of subsets of some set X made into a category, usually by defining a single abstract morphism Y→Z whenever Y⊆Z (this being a special case of the general method of treating pre-orders as categories). This allows a concise definition of presheaves and sheaves, and it is generalized when defining a site.
