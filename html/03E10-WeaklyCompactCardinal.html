weakly compact cardinal


Weakly compact cardinals are (large) infinite cardinals which have a property related to the syntactic compactness theorem for first order logic. Specifically, for any infinite cardinal κ, consider the language Lκ,κ.

This language is identical to first logic except that:

  • •

    infinite conjunctions and disjunctions of fewer than κ formulas are allowed

  • •

    infinite strings of fewer than κ quantifiers are allowed

The weak compactness theorem for Lκ,κ states that if Δ is a set of sentences of Lκ,κ such that |Δ|=κ and any θ⊂Δ with |θ|<κ is consistent then Δ is consistent.

A cardinal is weakly compact if the weak compactness theorem holds for Lκ,κ.
