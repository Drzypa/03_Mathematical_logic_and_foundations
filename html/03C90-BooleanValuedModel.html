Boolean valued model


A traditional model of a language makes every formula of that language either true or false. A Boolean valued model is a generalization in which formulas take on any value in a Boolean algebra.

Specifically, a Boolean valued model of a signature Σ over the language ℒ is a set 𝒜 together with a Boolean algebra ℬ. Then the objects of the model are the functions 𝒜ℬ=ℬ→𝒜.

For any formula ϕ, we can assign a value ∥ϕ∥ from the Boolean algebra. For example, if ℒ is the language of first order logic, a typical recursive definition of ∥ϕ∥ might look something like this:
