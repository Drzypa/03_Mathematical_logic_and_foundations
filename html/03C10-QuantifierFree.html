quantifier free


Let L be a first order language. A formula ψ is quantifier free iff it contains no quantifiers.

Let T be a complete L-theory. Let S⊆L. Then S is an elimination set for T iff for every ψ⁢(x¯)∈L there is some ϕ⁢(x¯)∈S so that T⊢∀x¯⁢(ψ⁢(x¯))↔ϕ⁢(x¯).

In particular, T has quantifier elimination iff the set of quantifier free formulas is an elimination set for T. In other T has quantifier elimination iff for every ψ⁢(x¯)∈L there is some quantifier free ϕ⁢(x¯)∈L so that T⊢∀x¯⁢(ψ⁢(x¯))↔ϕ⁢(x¯).
