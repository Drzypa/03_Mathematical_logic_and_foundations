ordinal exponentiation


Let α,β be ordinals. We define αβ as follows:

αβ:={1if ⁢β=0,αγ⋅αif β is a successor ordinal and ⁢β=S⁢γ,sup⁡{αγ∣γ<β}if β is a limit ordinal and ⁢β=sup⁡{γ∣γ<β}.

Some properties of exponentiation:

  1. 1.

    0α=0 if α>0

  2. 2.

    1α=1

  3. 3.

    α1=α

  4. 4.

    αβ⋅αγ=αβ+γ

  5. 5.

    (αβ)γ=αβ⋅γ

  6. 6.

    For any ordinals α,β with α>0 and β>1, there exists a unique triple (γ,δ,ϵ) of ordinals such that

    α=βγ⋅δ+ϵ

    where 0<δ<β and ϵ<βδ.

All of these properties can be proved using transfinite induction.
