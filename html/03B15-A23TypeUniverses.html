A.2.3 Type universes


We postulate an infinite hierarchy of type universes

𝒰0,𝒰1,𝒰2,…

Each universe is contained in the next, and any type in 𝒰i is also in 𝒰i+1: {mathpar} \inferrule*[right=𝒰-intro] Γ 𝖼𝗍𝗑 Γ ⊢𝒰 _i : 𝒰 _i+1 \inferrule*[right=𝒰-cumul] Γ ⊢A : 𝒰 _i Γ ⊢A : 𝒰 _i+1 We shall set up the rules of type theory in such a way that Γ⊢a:A implies Γ⊢A:𝒰i for some i. In other words, if A plays the role of a type then it is in some universe. Another property of our type system is that Γ⊢a≡b:A implies Γ⊢a:A and Γ⊢b:A.
