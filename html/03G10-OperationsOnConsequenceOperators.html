operations on consequence operators


Let L be a set and let 𝒞 be the set of all consequence operators on S. Then we may define a binary relation ≤⊂𝒞×𝒞 and binary operations ∧,∨,⊻:𝒞×𝒞→𝒞 as follows:

Definition 1

For C1,C2∈C, we have C1≤C2 when, for all X⊆L, we have C1⁢(X)⊆C2⁢(X)

Definition 2

For C1,C2∈C, we have (C1∧C2)⁢(X)=C1⁢(X)∩C2⁢(X) for all X⊆L.

Definition 3

For C1,C2∈C, we have (C1∨C2)⁢(X)=C1⁢(X)∪C2⁢(X) for all X⊆L.

Definition 4

For C1,C2∈C, we have (C1⊻C2)⁢(X)=∩{Y∣X⊆Y⊆L∧C1⁢(Y)=C2⁢(Y)=Y} for all X⊆L.
