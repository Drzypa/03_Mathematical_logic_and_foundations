theory


If L is a logical language for some logic ℒ, a set T of formulas with no free variables is called a theory (of ℒ). If ℒ is a first-order logic, then T is called a first-order theory.

We write T⊨ϕ for any formula ϕ if every model ℳ of ℒ such that M⊨T, M⊨ϕ.

We write T⊢ϕ is for there is a proof of ϕ from T.

Remark. Let S be an L-structure for some signature L. The theory of S is the set of formulas satisfied by S:

{φ∣S⊧φ},

and is denoted by Th⁡(S).
