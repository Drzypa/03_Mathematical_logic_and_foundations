inverse function theorem


Let 𝐟 be a continuously differentiable, vector-valued function mapping the open set E⊂ℝn to ℝn and let S=𝐟⁢(E). If, for some point 𝐚∈E, the Jacobian, |J𝐟⁢(𝐚)|, is non-zero, then there is a uniquely defined function 𝐠 and two open sets X⊂E and Y⊂S such that

  1. 1.

    𝐚∈X, 𝐟⁢(𝐚)∈Y;

  2. 2.

    Y=𝐟⁢(X);

  3. 3.

    𝐟:X→Y is one-one;

  4. 4.

    𝐠 is continuously differentiable on Y and 𝐠⁢(𝐟⁢(𝐱))=𝐱 for all 𝐱∈X.

0.0.1 Simplest case

When n=1, this theorem becomes: Let f be a continuously differentiable, real-valued function defined on the open interval I. If for some point a∈I, f′⁢(a)≠0, then there is a neighbourhood [α,β] of a in which f is strictly monotonic. Then y→f-1⁢(y) is a continuously differentiable, strictly monotonic function from [f⁢(α),f⁢(β)] to [α,β]. If f is increasing (or decreasing) on [α,β], then so is f-1 on [f⁢(α),f⁢(β)].

0.0.2 Note

The inverse function theorem is a special case of the implicit function theorem where the dimension of each variable is the same.
