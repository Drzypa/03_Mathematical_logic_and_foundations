quotient structure


Let Σ be a fixed signature, 𝔄 a structure for Σ, and ∼ a congruence on 𝔄. The quotient structure of 𝔄 by ∼, denoted 𝔄/∼, is defined as follows:

  1. 1.

    The universe of 𝔄/∼ is the set {[[a]]∣a∈𝔄}.

  2. 2.

    For each constant symbol c of Σ, c𝔄⁣/∼=[[c𝔄]].

  3. 3.

    For every natural number n and every n-ary function symbol F of Σ,

    F𝔄⁣/∼⁢([[a1]],…⁢[[an]])=[[F𝔄⁢(a1,…⁢an)]].
  4. 4.

    For every natural number n and every n-ary relation symbol R of Σ, R𝔄⁣/∼⁢([[a1]],…,[[an]]) if and only if for some ai′∼ai we have R𝔄⁢(a1′,…,an′).
