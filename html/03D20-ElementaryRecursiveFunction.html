elementary recursive function


Informally, elementary recursive functions are functions that can be obtained from functions encountered in elementary schools: addition, multiplication, subtraction, and division, using basic operations such as substitutions and finite summation and product. Before stating the formal definition, the following notations are used:

ℱ=⋃{Fk∣k∈ℕ}, where for each k∈ℕ⁢, ⁢Fk={f∣f:ℕk→ℕ}.

Definition. The set of elementary recursive functions, or elementary functions for short, is the smallest subset ℰ⁢ℛ of ℱ where:

  1. 1.

    (addition) add∈ℰ⁢ℛ∩F2, given by add⁡(m,n):=m+n;

  2. 2.

    (multiplication) mult∈ℰ⁢ℛ∩F2, given by mult⁡(m,n):=m⁢n;

  3. 3.

    (difference) diff∈ℰ⁢ℛ∩F2, given by diff⁡(m,n):=|m-n|;

  4. 4.

    (quotient) quo∈ℰ⁢ℛ∩F2, given by quo⁡(m,n):=[m/n], which is the largest non-negative integer z such that n⁢z≤m (by convention, quo⁡(0,0):=1);

  5. 5.

    (projection) pmk∈ℰ⁢ℛ∩Fk, where m≤k, given by pmk⁢(n1,…,nk):=nm;

  6. 6.

    ℰ⁢ℛ is closed under composition: If {g1,…,gm}⊆ℰ⁢ℛ∩Fk and h∈ℰ⁢ℛ∩Fm, then f∈ℰ⁢ℛ∩Fk, where

    f⁢(n1,…,nk)=h⁢(g1⁢(n1,…,nk),…,gm⁢(n1,…,nk));
  7. 7.

    ℰ⁢ℛ is closed under bounded sum: if f∈ℰ⁢ℛ∩Fk, then fs∈ℰ⁢ℛ∩Fk, where

    fs⁢(𝒙,y):=∑i=0yf⁢(𝒙,i);
  8. 8.

    ℰ⁢ℛ is closed under bounded product: if f∈ℰ⁢ℛ∩Fk, then fp∈ℰ⁢ℛ∩Fk, where

    fp⁢(𝒙,y):=∏i=0yf⁢(𝒙,i).

Examples.

  • •

    The initial functions in the definition of primitive recursive functions are elementary:

    1. (a)

      The zero function z⁢(x) is diff⁡(x,x).

    2. (b)

      The constant function const1⁡(x):=1 is quo⁡(x,x).

    3. (c)

      The successor function s⁢(x) can be obtained by substituting (by composition) the constant function const1 and the projection function p11, into the addition function add⁡(p11⁢(x),const1⁡(x)).

  • •

    Multiplication mult in 2 above may be removed from the definition, since

    mult⁡(x,y)=diff⁡(f⁢(x,y),p12⁢(x,y)),where ⁢f⁢(x,y):=∑i=0yp12⁢(x,i)
  • •

    Many other basic functions, such as the restricted subtraction, exponential function, the i-th prime function, are all elementary. One may replace the difference function in 3 above by the restricted subtraction without changing ℰ⁢ℛ.

Remarks

  • •

    Consider the set 𝒫⁢ℛ of primitive recursive functions. The functions in the first five groups above are all in 𝒫⁢ℛ. In addition, 𝒫⁢ℛ is closed under the operations in 6, 7, and 8 above, we see that ℰ⁢ℛ⊆𝒫⁢ℛ, since ℰ⁢ℛ, as defined, is the smallest such set.

  • •

    Furthermore, ℰ⁢ℛ≠𝒫⁢ℛ. For example, the super-exponential function, given by f⁢(x,0)=m, and f⁢(x,n+1)=exp⁡(m,f⁢(x,n)), where m>1, can be shown to be non-elementary.

  • •

    In addition, it can be shown that ℰ⁢ℛ is the set of primitive recursive functions that can be obtained from the zero function, the successor function, and the projection functions via composition, and no more than three applications of primitive recursion.

  • •

    By taking the closure of ℰ⁢ℛ with respect to unbounded minimization, one obtains ℛ, the set of all recursive functions (partial or total). In fact, by replacing bounded sum and bounded product with unbounded minimization, and the difference function with restricted subtraction, one obtains ℛ.
