some valid schemas of first order logic


In this entry, we record some valid schemas of first order logic FO(Σ) based on the signature Σ:

  1. 1.

    ∀x⁢A→∀y⁢A⁢[y/x] if y does not occur free, but free for x, in A

  2. 2.

    ∀y⁢A⁢[y/x]→∀x⁢A if y does not occur in A

  3. 3.

    ∀x⁢A↔A if x is not free in A

  4. 4.

    ∃x⁢A↔A if x is not free in A

  5. 5.

    ∀x⁢∀y⁢A→∀y⁢∀x⁢A

  6. 6.

    ∃x⁢∃y⁢A→∃y⁢∃x⁢A

  7. 7.

    ∀x(A→B)→(∀xA→∀xB)

  8. 8.

    (∀xA→∀xB)→∀x(A→B)

  9. 9.

    ∀x⁢(A∧B)↔∀x⁢A∧∀x⁢B

  10. 10.

    ∃x⁢(A∨B)↔∃x⁢A∨∃x⁢B

  11. 11.

    ∀x⁢(A∨B)↔(∀x⁢A)∨B if x is not free in B

  12. 12.

    ∃x⁢(A∧B)↔(∃x⁢A)∧B if x is not free in B

where A,B are well-formed formulas (wff’s) of FO(Σ).
