arithmetical hierarchy is a proper hierarchy


By definition, we have Δn=Πn∩Σn. In addition, Σn∪Πn⊆Δn+1.

This is proved by vacuous quantification. If R is equivalent to ϕ⁢(n→) then R is equivalent to ∀x⁢ϕ⁢(n→) and ∃x⁢ϕ⁢(n→), where x is some variable that does not occur free in ϕ.

More significant is the proof that all containments are proper. First, let n≥1 and U be universal for 2-ary Σn relations. Then D⁢(x)↔U⁢(x,x) is obviously Σn. But suppose D∈Δn. Then D∈P⁢in, so ¬⁢D∈Σn. Since U is universal, ther is some e such that ¬⁢D⁢(x)↔U⁢(e,x), and therefore ¬⁢D⁢(e)↔U⁢(e,e)↔¬⁢U⁢(e,e). This is clearly a contradiction, so D∈Σn∖Δn and ¬⁢D∈Πn∖Δn.

In addition the recursive join of D and ¬⁢D, defined by

D⊕¬D(x)↔(∃y<x[x=2⋅y]∧D(x))∨(¬∃y<x[x=2⋅y]∧¬D(x))

Clearly both D and ¬⁢D can be recovered from D⊕¬⁢D, so it is contained in neither Σn nor Πn. However the definition above has only unbounded quantifiers except for those in D and ¬⁢D, so D⊕¬⁢D⁢(x)∈Δn+1∖Σn∪Πn
