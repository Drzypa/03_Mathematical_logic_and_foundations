Presburger arithmetic


Presburger arithmetic is a weakened form of arithmetic which includes the structure ℕ, the constant 0, the unary function S, the binary function +, and the binary relation <. Essentially, it is Peano arithmetic without multiplication.

The axioms are:

  1. 1.

    0≠S⁢x

  2. 2.

    S⁢x=S⁢y→x=y

  3. 3.

    x+0=x

  4. 4.

    x+S⁢y=S⁢(x+y)

  5. 5.

    For each first order formula P⁢(x), P(0)∧∀x[P(x)→P(x+1)]→∀xP(x)

Presburger arithmetic is decidable, but is consequently very limited in what it can express.
