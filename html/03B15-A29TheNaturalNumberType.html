A.2.9 The natural number type


We give the rules for natural numbers, following §1.9 (http://planetmath.org/19thenaturalnumbers).

{mathparpagebreakable}\inferrule

*[right=ℕ-form] Γ 𝖼𝗍𝗑 Γ ⊢ℕ : 𝒰 _i \inferrule*[right=ℕ-intro1] Γ 𝖼𝗍𝗑 Γ ⊢0 : ℕ \inferrule*[right=ℕ-intro2] Γ ⊢n : ℕ Γ ⊢succ(n) : ℕ \inferrule*[right=ℕ-elim] Γ,x : ℕ ⊢C : 𝒰 _i
Γ ⊢c_0 : C[0/x]
Γ,x : ℕ,y : C ⊢c_s : C[succ(x)/x]
Γ ⊢n : ℕ Γ ⊢ind_ℕ(x.C,c_0,x.y.c_s,n) : C[n/x] \inferrule*[right=ℕ-comp1] Γ,x : ℕ ⊢C : 𝒰 _i
Γ ⊢c_0 : C[0/x]
Γ,x : ℕ,y : C ⊢c_s : C[succ(x)/x] Γ ⊢ind_ℕ(x.C,c_0,x.y.c_s,0) ≡c_0 : C[0/x] \inferrule*[right=ℕ-comp2] Γ,x : ℕ ⊢C : 𝒰 _i
Γ ⊢c_0 : C[0/x]
Γ,x : ℕ,y : C ⊢c_s : C[succ(x)/x]
Γ ⊢n : ℕ Γ⊢ ind ℕ (x.C,c 0 ,x.y.c s ,succ(n))  ≡c s [n,ind ℕ (x.C,c 0 ,x.y.c s ,n)/x,y] : C[succ(n)/x] In 𝗂𝗇𝖽ℕ, x is bound in C, and x and y are bound in cs.

Other inductively defined types follow the same general scheme.
