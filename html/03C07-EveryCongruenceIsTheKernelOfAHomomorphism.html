every congruence is the kernel of a homomorphism


Let Σ be a fixed signature, and 𝔄 a structure for Σ. If ∼ is a congruence on 𝔄, then there is a homomorphism f such that ∼ is the kernel of f.

Proof.

Define a homomorphism f:𝔄→𝔄/∼:a↦[[a]]. Observe that a∼b if and only if f⁢(a)=f⁢(b), so ∼ is the kernel of f. To verify that f is a homomorphism, observe that

  1. 1.

    For each constant symbol c of Σ, f⁢(c𝔄)=[[c𝔄]]=c𝔄⁣/∼.

  2. 2.

    For each n∈ℕ and each n-ary function symbol F of Σ,

    f⁢(F𝔄⁢(a1,…⁢an)) =[[F𝔄⁢(a1,…⁢an)]]
    =F𝔄⁣/∼⁢([[a1]],…⁢[[an]])
    =F𝔄⁣/∼⁢(f⁢(a1),…⁢f⁢(an)).∎
  3. 3.

    For each n∈ℕ and each n-ary relation symbol R of Σ, if R𝔄⁢(a1,…,an) then R𝔄⁣/∼⁢([[a1]],…,[[an]]), so R𝔄⁣/∼⁢(f⁢(a1),…,f⁢(an)).
