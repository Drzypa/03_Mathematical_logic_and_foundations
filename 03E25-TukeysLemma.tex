Each nonempty family of finite character has a maximal element.

Here, by a maximal element we \PMlinkescapetext{mean} a maximal element with respect to the inclusion ordering: $A\leq B$ iff $A\subseteq B$. This lemma is equivalent to the axiom of choice.
